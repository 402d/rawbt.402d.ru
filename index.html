Very small letters on the printout.


The page that you want to print, the program receives in graphical format (picture). If it was created for A4 paper, then the reduction occurs 4 times. Use a larger font in your document. On the print driver side, font enlargement is not possible. On this site you will find descriptions of all the ways to interact with the program to get the best result.

Printing is easy!

To print the text to thermal printer connected to client's phone it is enough to place a special type of link on the site.
